# 📝 Notes App (Flutter)

A simple and efficient **Notes Application** built using **Flutter**.  
The app allows users to create, edit, and delete notes while storing data locally on the device.

This project demonstrates **state management using Cubit** and **local storage using Hive** to build a fast and responsive notes application.

---

## 🚀 Features

- Add new notes
- Edit existing notes
- Delete notes
- Store notes locally on the device
- Fast data access using Hive database
- Reactive UI updates using Cubit state management
- Simple and clean user interface

---

## 🛠️ Technologies Used

- Flutter
- Cubit (Bloc State Management)
- Hive (Local Database)

---


---

## 🎯 Project Goal

The goal of this project is to practice:

- Flutter UI development
- State management using Cubit
- Local data persistence using Hive
- Building a clean and maintainable Flutter application
